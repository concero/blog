## Introduction

Concero is developing an abstracted interoperability platform for future cross-chain applications. A decentralised infrastructure which enables cross-chain transactions in under a minute without compromising security. With gas abstraction, routing, and aggregation, Concero provides developers with essential tools for building secure, fast, and user-friendly cross-chain apps. Offering an unmatched tradeoff between speed and security, surpassing competitors. With a fully decentralised infrastructure, Concero stands out as the only truly decentralised stack for development.

## How to Test Concero's CCIP Testnet? 

_We recommend you to only use MetaMask Desktop version for now as Mobile isn't supported yet_

1. Head over to app.concero.io

![Concero Public Testnet Launch](/static/images/concero/announcement/concero-public-testnet-launch/fig1.webp)

2. Connect your wallet with the provided wallet address for the testnet signup application.

![Concero Public Testnet Launch](/static/images/concero/announcement/concero-public-testnet-launch/fig2.webp)

3. Once you have connected your wallet to app.concero.io make sure to be on any of the three test networks, for example we're on Optimism Sepolia now, you can add test networks here https://chainlist.org/chain/11155420

![Concero Public Testnet Launch](/static/images/concero/announcement/concero-public-testnet-launch/fig3.webp)

4. Click on Get testnet ETH & LINK, which will redirect you to https://faucets.chain.link/

Click on connect wallet and connect your wallet

**Make sure that you are going to get the tokens on the same network on which you are going to make the transaction**

![Concero Public Testnet Launch](/static/images/concero/announcement/concero-public-testnet-launch/fig4.webp)

5. After which you can request for 25 test LINK on Optimism Sepolia (Can claim 0.25 test ETH if you have a Github account as well)

![Concero Public Testnet Launch](/static/images/concero/announcement/concero-public-testnet-launch/fig5.webp)

6. You would receive the test tokens in your wallet

![Concero Public Testnet Launch](/static/images/concero/announcement/concero-public-testnet-launch/fig6.webp)

7. Head back over to app.concero.io and click on get CCIPBnM tokens, for which you would have to pay for gas with ETH on Optimism Sepolia, if you are low on ETH for gas fees you can request it from (https://www.ethereum-ecosystem.com/faucets/base-sepolia) or (https://www.ethereum-ecosystem.com/faucets/optimism-sepolia)

![Concero Public Testnet Launch](/static/images/concero/announcement/concero-public-testnet-launch/fig7.webp)
