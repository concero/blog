## Introduction

Concero is developing an abstracted interoperability platform for future cross-chain applications. A decentralised infrastructure which enables cross-chain transactions in under a minute without compromising security. With gas abstraction, routing, and aggregation, Concero provides developers with essential tools for building secure, fast, and user-friendly cross-chain apps. Offering an unmatched tradeoff between speed and security, surpassing competitors. With a fully decentralised infrastructure, Concero stands out as the only truly decentralised stack for development.

## How to Test Concero's CCIP Testnet? 

_We recommend you to only use MetaMask Desktop version for now as Mobile isn't supported yet_

1. Head over to app.concero.io

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig1.webp)

2. Connect your wallet with the provided wallet address for the testnet signup application.

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig2.webp)

3. Once you have connected your wallet to app.concero.io make sure to be on any of the three test networks, for example we're on Base Sepolia now, you can add test networks here https://chainlist.org/chain/11155420

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig3.webp)

4. Click on Get testnet ETH & LINK, which will redirect you to https://faucets.chain.link/

Click on connect wallet and connect your wallet

**Make sure that you are going to get the tokens on the same network on which you are going to make the transaction**

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig4.webp)

5. After which you can request for 25 test LINK on Base Sepolia (Can claim 0.25 test ETH if you have a Github account as well)

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig5.webp)

6. You would receive the test tokens in your wallet

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig6.webp)

7. Head back over to app.concero.io and click on get CCIPBnM tokens, for which you would have to pay for gas with ETH on Base Sepolia, if you are low on ETH for gas fees you can request it from (https://www.ethereum-ecosystem.com/faucets/base-sepolia) or (https://www.ethereum-ecosystem.com/faucets/optimism-sepolia)

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig7.webp)

8. The test tokens we'll be bridging for this instance will be CCIP-BnM please make sure to add/import both Test LINK and CCIP-BnM to your metamask wallet tokens before swapping

- CCIPBNM ARBITRUM SEPOLIA 0xA8C0c11bf64AF62CDCA6f93D3769B88BdD7cb93D
- CCIPBNM BASE SEPOLIA 0x88A2d74F47a237a62e7A51cdDa67270CE381555e
- CCIPBNM OPTIMISM SEPOLIA 0x8aF4204e30565DF93352fE8E1De78925F6664dA7

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig8.webp)

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig9.webp)

- LINK ARBITRUM SEPOLIA 0xE4aB69C077896252FAFBD49EFD26B5D171A32410
- LINK BASE SEPOLIA 0xE4aB69C077896252FAFBD49EFD26B5D171A32410
- LINK OPTIMISM SEPOLIA 0xE4aB69C077896252FAFBD49EFD26B5D171A32410

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig10.webp)

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig11.webp)

9. Please make sure to swap equal to lesser than 0.1

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig12.webp)

10. Approve Test Link and CCIP-BnM and confirm the transaction

(Please make sure set the site estimate gas fees or else your txns may fail and also ensure you have both Test LINK and CCIP-BnM tokens in your wallet before swapping)

![Concero Testnet Live](/static/images/concero/announcement/concero-testnet-live/fig13.webp)

11. Congratulations you've completed Concero's CCIP Testnet txn, You could play around with other chains the same way.
